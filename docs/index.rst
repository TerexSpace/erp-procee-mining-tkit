ERP-ProcessMiner Documentation
===============================

Welcome to the documentation for ``ERP-ProcessMiner``, a Python toolkit for process mining on ERP event logs.

This toolkit is designed to help researchers, teachers, and analysts bridge the gap between raw, relational ERP data and the structured event logs required for process mining.
